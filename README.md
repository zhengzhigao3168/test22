# 图片采集程序使用说明

## 文件说明
- `edotfish图像处理升级版.py`: 主程序文件
- `alibaba_product_info.py`: 阿里巴巴产品信息处理模块
- `requirements.txt`: 依赖包列表
- `frontend_config.json`: 前端配置文件
- `standalone_api.py`: 独立API服务
- `minimal_api.py`: 最小API服务
- `run_backend.py`: 后端服务启动脚本
- `run_frontend.py`: 前端服务启动脚本
- `start.bat`: 快速启动脚本

## 使用方法
1. 确保已安装Python环境（建议Python 3.8或更高版本）
2. 双击运行`start.bat`，程序会自动安装所需依赖并启动
3. 按照程序界面提示进行操作

## 注意事项
- 首次运行时需要联网安装依赖包
- 请确保有足够的磁盘空间用于存储图片
- 建议使用稳定的网络连接

## 常见问题
如果遇到问题，请检查：
1. Python环境是否正确安装
2. 网络连接是否正常
3. 依赖包是否完整安装

## 技术支持
如有问题请参考程序内的帮助文档或联系技术支持。
